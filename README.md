# NodeJS REST API + TypeScripts

Hanya untuk belajar :) Have Fun

## Package
- Typescripts
- Nodemon
- ExpressJS
- Mongoose
- Bcrypt
- JSON Web Token

## Install
```bash
yarn install
```

## Development
```bash
yarn tsc
yard dev
```
